# r-sugar
Personal R helper functions

### Usage
```
devtools::install_github('highandmighty/r-sugar')
library(sugarr)

> ?fetch_airtable
> ?download_feather
```

### Functions
- `download_feather()` — fetches Feather file from URL into dataframe
- `fetch_airtable()` — fetches data from Airtable API into dataframe
